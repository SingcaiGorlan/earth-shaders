# Halo轨道和环月冻结轨道 - 快速参考

## 🚀 快速启动

1. 访问 `http://localhost:5174/src/launch.html`
2. 在"物理模拟"区域勾选 **"拉格朗日点 & 特殊轨道"**
3. 点击"启动应用"

## 📍 可视化元素说明

### 拉格朗日点标记
- 🔴 **红色球体**：L1, L2, L3（不稳定）
- 🟢 **绿色球体**：L4, L5（稳定）
- 白色文字标签：L1-L5

### Halo轨道
- 🟠 **橙色线条**：围绕L1和L2的三维晕轨道
- 形状：8字形或椭圆形
- 动画：缓慢旋转展示三维结构

### 环月冻结轨道
- 🟩 **青绿色** (27°倾角) - 低倾角观测
- 🟨 **黄绿色** (50°倾角) - 中纬度观测
- 🟪 **紫红色** (76°倾角) - 极地观测
- 🟧 **橙黄色** (86°倾角) - 近极地全球覆盖
- 🔴 **红色** (大椭圆) - 50-500km高度变化

## 🎯 关键概念

### 拉格朗日点
地月系统中引力平衡的5个特殊位置，航天器可在此保持相对静止。

### Halo轨道
围绕L1/L2点的三维周期性轨道，用于深空观测和中继通信。

### 冻结轨道
月球特殊椭圆轨道，偏心率保持不变，适合长期月球探测任务。

## 💡 观察技巧

1. **缩放查看**：滚轮缩放到月球附近查看冻结轨道细节
2. **旋转视角**：右键拖动从不同角度观察Halo轨道的三维结构
3. **对比颜色**：不同颜色的轨道代表不同的任务需求

## 🔬 实际应用

| 轨道类型 | 著名任务 |
|---------|---------|
| L2 Halo | JWST望远镜、鹊桥中继星 |
| L1 Halo | DSCOVR太阳监测卫星 |
| 冻结轨道 | LRO月球勘测、嫦娥系列 |

- **坐标系**：地月质心坐标系
- **比例**：真实距离按比例缩放
