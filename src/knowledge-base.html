1. 轨道力学基础

1.1 开普勒定律

第一定律（椭圆轨道定律）

所有行星绕太阳运动的轨道都是椭圆，太阳位于椭圆的一个焦点上。

r = a(1 - e²) / (1 + e·cos(ν))

应用：人造卫星绕地球运动同样遵循此定律。

第二定律（面积速度定律）

行星在相等时间内扫过的面积相等。

dA/dt = constant = h/2

推论：近地点速度快，远地点速度慢。

第三定律（周期定律）

轨道周期的平方与半长轴的立方成正比。

T² = (4π²/GM) × a³

简化：对于地球轨道，当T以年为单位，a以AU为单位时：T² = a³

1.2 轨道六根数（Keplerian Elements）

参数 符号 说明 单位
半长轴 a 轨道大小 km
偏心率 e 轨道形状（0=圆，0<e<1=椭圆） 无量纲
轨道倾角 i 轨道平面与赤道面夹角 °
升交点赤经 Ω 升交点相对于春分点的角度 °
近地点幅角 ω 近地点相对于升交点的角度 °
真近点角 ν 卫星当前位置相对于近地点的角度 °

1.3 特征速度计算

圆周轨道速度

v_circular = √(μ/r)

其中 μ = GM = 398600.4418 km³/s²（地球引力常数）

逃逸速度

v_escape = √(2μ/r) = √2 × v_circular ≈ 1.414 × v_circular

霍曼转移所需ΔV

ΔV₁ = √(μ/r₁) × (√(2r₂/(r₁+r₂)) - 1)
ΔV₂ = √(μ/r₂) × (1 - √(2r₁/(r₁+r₂)))
总ΔV = ΔV₁ + ΔV₂

2. 火箭推进原理

2.1 齐奥尔科夫斯基火箭方程

基本形式

ΔV = Isp × g₀ × ln(m₀/mf)

ΔV = 速度增量（m/s）

Isp = 比冲（s）

g₀ = 标准重力加速度 = 9.80665 m/s²

m₀ = 初始质量（含燃料）

mf = 最终质量（燃料耗尽后）

质量比

MR = m₀/mf = exp(ΔV / (Isp × g₀))

2.2 推力计算

基本推力公式

F = ṁ × Ve + (Pe - Pa) × Ae

真空简化：F = ṁ × Ve

F = 推力（N）

ṁ = 质量流量（kg/s）

Ve = 排气速度（m/s）

2.3 比冲（Specific Impulse）

定义

单位重量推进剂产生的冲量，衡量发动机效率的关键指标。

Isp = F / (ṁ × g₀) = Ve / g₀

典型发动机比冲对比

发动机类型 Isp（海平面） Isp（真空） 应用
固体火箭 250-290 s 280-300 s 助推器
煤油/液氧 280-310 s 330-350 s 一级火箭
液氢/液氧 360-390 s 440-465 s 上面级
离子推进器 - 2000-5000 s 深空探测

2.4 TWR（推重比）

定义

TWR = F / (m × g)

关键阈值

  • TWR > 1：能够起飞
  • TWR = 1.5-2.5：理想的发射推重比
  • TWR < 1：无法离地

随时间变化：发射时TWR最小（质量最大），燃料消耗后TWR增加。需要注意过高的TWR可能导致结构过载或大气阻力过大。

3.2 空气动力学基础

升力公式

L = ½ × ρ × v² × S × CL

阻力公式

D = ½ × ρ × v² × S × CD

动压（Dynamic Pressure）

q = ½ × ρ × v²

Max-Q：最大动压点，通常在10-15 km高度，是火箭设计中的关键参数。

3.3 重力转弯（Gravity Turn）

最优策略

  1. 垂直上升至100-200 m/s
  2. 开始小幅倾斜（5-10°）
  3. 让重力逐渐增加倾角
  4. 到达目标高度时达到水平

优点

  • ✅ 节省燃料（相比纯垂直+纯水平）
  • ✅ 减少结构应力
  • ✅ 自动控制简单

4. 姿态控制与稳定

4.1 坐标系定义

机体坐标系（Body Frame）

  • X轴：纵向（向前为正）
  • Y轴：横向（向右为正）
  • Z轴：垂直（向下为正，右手定则）

惯性坐标系（Inertial Frame）

  • 原点：地心
  • XY平面：赤道面
  • X轴：指向春分点

4.2 欧拉角与四元数

三个旋转角

  1. 滚转（Roll, φ）：绕X轴旋转
  2. 俯仰（Pitch, θ）：绕Y轴旋转
  3. 偏航（Yaw, ψ）：绕Z轴旋转

⚠️ 注意万向节锁（Gimbal Lock）问题

四元数（Quaternion）

q = w + xi + yj + zk
其中 w² + x² + y² + z² = 1

优势

  • ✅ 避免万向节锁
  • ✅ 计算效率高
  • ✅ 插值平滑（SLERP）

4.3 姿态控制方法

反作用轮（Reaction Wheel）

通过改变轮的转速产生扭矩

  • ✓ 精度高
  • ✓ 适合精细控制
  • ✗ 会饱和，需要卸载

控制力矩陀螺（CMG）

高速旋转的陀螺仪

  • ✓ 大扭矩
  • ✓ 适合大型航天器
  • ✗ 复杂机械结构

推力器（Thruster）

RCS反应控制系统

  • ✓ 快速响应
  • ✓ 适用范围广
  • ✗ 消耗推进剂

气动舵面

副翼、升降舵、方向舵

  • ✓ 无燃料消耗
  • ✓ 成熟技术
  • ✗ 仅在大气层内有效

5. 重力与引力场

5.1 牛顿万有引力定律

基本公式

F = G × (m₁ × m₂) / r²

G = 6.67430 × 10⁻¹¹ m³/(kg·s²)

m₁, m₂ = 两物体质量

r = 距离

重力加速度

g = GM / r²

对于地球：GM = μ = 398600.4418 km³/s²，地表g₀ = 9.80665 m/s²

5.2 重力随高度变化

精确公式

g(h) = g₀ × (R / (R + h))²

其中 R = 地球半径 = 6371 km，h = 高度

高度 重力加速度 相对地表
0 km 9.80665 m/s² 100%
100 km（卡门线） 9.51 m/s² 97%
400 km（ISS） 8.69 m/s² 89%
35786 km（GEO） 0.224 m/s² 2.3%

5.3 拉格朗日点

点 位置 稳定性 应用
L1 两天体之间 不稳定 太阳观测卫星
L2 外侧 不稳定 JWST望远镜
L3 对面 不稳定 理论研究
L4 前方60° 稳定 特洛伊小行星
L5 后方60° 稳定 特洛伊小行星

6. 发射窗口与轨道转移

6.1 霍曼转移（Hohmann Transfer）

原理

最节能的双脉冲轨道转移，适用于共面圆轨道之间的转移。

所需ΔV

ΔV₁ = √(μ/r₁) × (√(2r₂/(r₁+r₂)) - 1)
ΔV₂ = √(μ/r₂) × (1 - √(2r₁/(r₁+r₂)))

转移时间

t_transfer = π × √(a_transfer³ / μ)

示例：LEO到GEO

  • r₁ = 6571 km（200 km高度）
  • r₂ = 42164 km（GEO）
  • ΔV₁ ≈ 2.44 km/s
  • ΔV₂ ≈ 1.47 km/s
  • 总ΔV ≈ 3.91 km/s
  • 转移时间 ≈ 5.3小时

6.2 轨道平面变更

所需ΔV

ΔV = 2 × v × sin(Δi/2)

其中 v = 当前轨道速度，Δi = 倾角变化量

关键点

  • ⚠️ 在速度最低点进行（远地点）
  • ⚠️ 60°以上的平面变更极其昂贵
  • ⚠️ 最好从发射时就选择正确倾角

示例

LEO速度：7.8 km/s

  • 改变10°倾角：ΔV ≈ 1.36 km/s
  • 改变28.5°倾角：ΔV ≈ 3.82 km/s（几乎等于整个轨道速度！）

7. 热力学与再入物理

7.1 再入加热机制

主要热源

  1. 激波压缩（占90%以上）：高速气体被压缩升温，形成高温等离子体
  2. 摩擦加热：边界层内的粘性耗散，贡献较小

驻点温度估算

T_stagnation ≈ T_∞ × (1 + (γ-1)/2 × M²)

其中 γ = 比热比（空气≈1.4），M = 马赫数

典型值

  • 返回舱再入：M ≈ 25
  • 激波温度：> 10000 K
  • 表面温度：1500-3000°C

7.2 热防护系统（TPS）

烧蚀材料

通过材料分解带走热量

  • ✓ 一次性使用
  • ✓ 例：阿波罗指令舱、神舟飞船

辐射冷却

高温表面辐射散热

  • ✓ 可重复使用
  • ✓ 例：航天飞机隔热瓦

主动冷却

内部流体循环

  • ✓ 复杂但高效
  • ✓ 例：火箭发动机喷管

7.3 黑障现象

成因

高温电离气体形成等离子体鞘，阻挡无线电波传播。

影响

  • 持续时间：典型3-5分钟
  • 高度范围：80-40 km
  • 影响频率：UHF/VHF完全阻断，更高频率部分衰减

8. 导航制导与控制

8.1 惯性导航系统（INS）

原理

使用加速度计和陀螺仪，积分得到速度和位置。

误差累积

位置误差随时间增长，典型漂移：1-10 km/hour

组件

  1. 加速度计：测量比力（specific force），需要减去重力
  2. 陀螺仪：测量角速度，积分得到姿态

8.2 GPS导航

精度

  • 民用：5-10 m
  • 军用：1 m以内
  • RTK：厘米级

局限性

  • ⚠️ 需要视线（LOS）
  • ⚠️ 高动态环境可能失锁
  • ⚠️ 易受干扰

8.3 制导律

比例导引（Proportional Navigation）

a_command = N × Vc × λ̇

N = 导航常数（3-5）

Vc = 接近速度

λ̇ = 视线角速率

应用：导弹制导、交会对接、着陆引导

PID控制

u(t) = Kp×e(t) + Ki×∫e(t)dt + Kd×de/dt
  • P：比例项（当前误差）
  • I：积分项（消除稳态误差）
  • D：微分项（预测趋势）

8.4 自主导航

星敏感器

拍摄星空图像，识别恒星图案

  • ✓ 确定绝对姿态
  • ✓ 精度：弧秒级

地平仪

检测地球边缘

  • ✓ 提供俯仰/滚转参考
  • ✓ 简单可靠

激光雷达（LiDAR）

近距离高精度测距

  • ✓ 用于着陆和交会
  • ✓ 例：SpaceX Dragon对接

⚠️ 易错点提醒

常见错误

  • ❌ 单位混淆：r是距地心距离，不是高度。正确：r = R_earth + h
  • ❌ 公式误用：逃逸速度是圆周速度的√2倍，不是2倍
  • ❌ 概念混淆：Isp（比冲）vs TWR（推重比）
  • ❌ 忽略条件：轨道平面变更应在远地点进行
