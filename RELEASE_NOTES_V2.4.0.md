## 🎉 新功能

### 🚀 航天器发射模拟器 (Spacecraft Launcher)

全新添加的航天器发射和轨道模拟系统，实现从地球表面任意位置发射航天器，并在真实物理引力作用下进行轨道运行。

#### 核心特性

1. **物理引擎**
   - ✅ 基于牛顿万有引力定律的真实物理计算
   - ✅ RK4四阶龙格-库塔法高精度数值积分
   - ✅ 支持多体引力（当前为地球，可扩展月球、太阳）
   - ✅ 实时加速度、速度、位置计算

2. **发射系统**
   - ✅ 地理坐标系统支持（纬度、经度、高度）
   - ✅ 任意发射方向控制（方位角0-360°、仰角0-90°）
   - ✅ 发射速度范围：1000-15000 m/s
   - ✅ 自动坐标转换（地理坐标 → 笛卡尔坐标）

3. **可视化**
   - ✅ 绿色发光球形航天器模型
   - ✅ 青色半透明轨道轨迹线
   - ✅ 最多保留500个轨迹点
   - ✅ 实时状态显示（高度、速度、时间）

4. **控制系统**
   - ✅ GUI控制面板完整集成
   - ✅ 暂停/继续模拟功能
   - ✅ 时间缩放控制（0.1x - 10x）
   - ✅ 一键重置功能
   - ✅ 轨迹显示/隐藏切换

## 🎮 使用示例

### 基本用法

```javascript
// 1. 创建发射器实例
const launcher = new SpacecraftLauncher(scene, earthGravityModel);

// 2. 设置发射参数
launcher.setLaunchSite(28.5, -80.6, 0); // 肯尼迪航天中心
launcher.setLaunchVelocity(7800, 90, 0); // 7800 m/s，向东，水平

// 3. 发射
launcher.launch();

// 4. 在动画循环中更新
function animate() {
    const dt = clock.getDelta();
    launcher.update(dt);
}
```

### 预设场景

1. **近地轨道 (LEO)**: 速度7800 m/s，仰角0°，方位角90°
2. **垂直发射**: 速度3000 m/s，仰角90°
3. **逃逸轨道**: 速度11200 m/s，高度200km
4. **极地轨道**: 速度7600 m/s，方位角0°或180°

## 🔧 技术细节

### 物理模型

**引力加速度**:
```
a = -GM/r² × r̂

G = 6.67430×10⁻¹¹ m³/(kg·s²)
M = 5.9722×10²⁴ kg (地球质量)
r = 距离地心的距离
```

**RK4积分公式**:
```
k1 = f(t, y)
k2 = f(t + dt/2, y + k1×dt/2)
k3 = f(t + dt/2, y + k2×dt/2)
k4 = f(t + dt, y + k3×dt)

y(t+dt) = y(t) + (k1 + 2k2 + 2k3 + k4) × dt/6
```

### 坐标系统

**地理坐标 → 场景坐标**:
```javascript
φ' = (90 - φ) × π/180  // 余纬度
λ' = (λ + 180) × π/180  // 经度弧度
r = R_earth × (1 + h/R_earth)

x = r × sin(φ') × cos(λ')
y = r × cos(φ')
z = r × sin(φ') × sin(λ')
```

## 🐛 已知问题

1. **仅支持地球引力**: 当前版本只实现了地球引力模型，月球和太阳引力待扩展
2. **无大气阻力**: 未实现大气层内的空气阻力计算
3. **无推进系统**: 不支持中途变轨和发动机点火
4. **轨迹点限制**: 最多500个点，长时间模拟可能丢失早期轨迹
