## 1. 轨道力学基础

> **学习目标**：掌握开普勒定律、轨道六根数、特征速度计算，能够分析卫星轨道特性。

### 1.1 开普勒三大定律

#### 【概念定义】
描述天体运动的基本规律，是轨道力学的理论基础。

#### 【关键公式/原理】

**第一定律（椭圆轨道定律）**
- 所有行星绕太阳运动的轨道都是椭圆，太阳位于椭圆的一个焦点上
- 轨道方程：`r = a(1 - e²) / (1 + e·cos(ν))`
- 应用于人造卫星：地球位于轨道的一个焦点

**第二定律（面积速度定律）**
- 行星在相等时间内扫过的面积相等
- 公式：`dA/dt = constant = h/2`（h为角动量）
- **推论**：近地点速度快，远地点速度慢

**第三定律（周期定律）**
- 轨道周期的平方与半长轴的立方成正比
- 公式：`T² = (4π²/GM) × a³`
- 对于地球轨道简化：`T² = a³`（T以年为单位，a以AU为单位）

#### 【典型应用场景】
- ✅ 计算卫星轨道周期
- ✅ 预测卫星位置
- ✅ 设计霍曼转移轨道
- ✅ 分析行星际探测任务

---

### 1.2 轨道六根数（Keplerian Elements）

#### 【概念定义】
描述轨道形状、大小、空间位置和卫星位置的六个独立参数。

#### 【关键参数】

| 参数 | 符号 | 物理意义 | 单位 |
|------|------|---------|------|
| 半长轴 | a | 轨道大小 | km |
| 偏心率 | e | 轨道形状（0=圆，0<e<1=椭圆） | 无量纲 |
| 轨道倾角 | i | 轨道平面与赤道面夹角 | ° |
| 升交点赤经 | Ω | 升交点相对于春分点的角度 | ° |
| 近地点幅角 | ω | 近地点相对于升交点的角度 | ° |
| 真近点角 | ν | 卫星当前位置相对于近地点的角度 | ° |

#### 【典型应用场景】

### 1.3 轨道能量方程

#### 【概念定义】
描述轨道机械能守恒关系，用于判断轨道类型。

#### 【关键公式】

**比机械能（Specific Mechanical Energy）**
```
ε = v²/2 - μ/r
```
其中：
- v = 速度（km/s）
- μ = GM = 398600.4418 km³/s²（地球引力常数）
- r = 距地心距离（km）

**轨道类型判断**
- ε < 0：椭圆轨道（束缚轨道）
- ε = 0：抛物线轨道（逃逸轨道）
- ε > 0：双曲线轨道（超逃逸轨道）

#### 【典型应用场景】
- ✅ 判断探测器是否脱离地球引力
- ✅ 计算轨道转移所需能量
- ✅ 分析捕获轨道

---

### 1.4 特征速度计算

#### 【概念定义】
轨道力学中的关键速度指标，用于任务设计和燃料估算。

#### 【关键公式】

**圆周轨道速度**
```
v_circular = √(μ/r)
```

**逃逸速度**
```
v_escape = √(2μ/r) = √2 × v_circular ≈ 1.414 × v_circular
```

**霍曼转移所需ΔV**
```
ΔV₁ = √(μ/r₁) × (√(2r₂/(r₁+r₂)) - 1)
ΔV₂ = √(μ/r₂) × (1 - √(2r₁/(r₁+r₂)))
总ΔV = ΔV₁ + ΔV₂
```

#### 【典型应用场景】
- ✅ LEO卫星发射速度计算
- ✅ 深空探测器逃逸速度需求
- ✅ 轨道转移燃料预算
- ✅ 任务可行性分析

## 2. 火箭推进原理

### 2.1 齐奥尔科夫斯基火箭方程

**基本形式**
```
ΔV = Isp × g₀ × ln(m₀/mf)
```

其中：
- ΔV = 速度增量（m/s）
- Isp = 比冲（s）
- g₀ = 标准重力加速度 = 9.80665 m/s²
- m₀ = 初始质量（含燃料）
- mf = 最终质量（燃料耗尽后）
- ln = 自然对数

**质量比**
```
MR = m₀/mf = exp(ΔV / (Isp × g₀))
```

### 2.2 推力计算

**基本推力公式**
```
F = ṁ × Ve + (Pe - Pa) × Ae
```

简化形式（真空）：
```
F = ṁ × Ve
```

其中：
- F = 推力（N）
- ṁ = 质量流量（kg/s）
- Ve = 排气速度（m/s）
- Pe = 喷管出口压力（Pa）
- Pa = 环境压力（Pa）
- Ae = 喷管出口面积（m²）

**推力系数**
```
F = Cf × Pc × At
```
- Cf = 推力系数
- Pc = 燃烧室压力
- At = 喉部面积

### 2.3 比冲（Specific Impulse）

**定义**
- 单位重量推进剂产生的冲量
- 衡量发动机效率的关键指标

**计算公式**
```
Isp = F / (ṁ × g₀) = Ve / g₀
```

**典型值对比**

| 发动机类型 | Isp（海平面） | Isp（真空） | 应用 |
|-----------|--------------|------------|------|
| 固体火箭 | 250-290 s | 280-300 s | 助推器 |
| 煤油/液氧 | 280-310 s | 330-350 s | 一级火箭 |
| 液氢/液氧 | 360-390 s | 440-465 s | 上面级 |
| 离子推进器 | - | 2000-5000 s | 深空探测 |

### 2.4 多级火箭

**分级优势**
- 抛弃空燃料箱和发动机，减轻质量
- 每级使用优化的发动机

**总ΔV计算**
```
ΔV_total = Σ ΔVi = Σ (Ispi × g₀ × ln(m₀i/mfi))
```

**质量比优化**
- 理想情况下，各级质量比应相近
- 实际设计中考虑结构质量和有效载荷

### 2.5 TWR（推重比）

**定义**
```
TWR = F / (m × g)
```

**关键阈值**
- TWR > 1：能够起飞
- TWR = 1.5-2.5：理想的发射推重比
- TWR < 1：无法离地

**随时间变化**
- 发射时TWR最小（质量最大）
- 燃料消耗后TWR增加
- 需要注意过高的TWR可能导致结构过载或大气阻力过大

## 3. 大气飞行力学

### 3.1 大气模型

**国际标准大气（ISA）**

| 高度（km） | 温度（K） | 压力（Pa） | 密度（kg/m³） |
|-----------|----------|-----------|--------------|
| 0（海平面） | 288.15 | 101325 | 1.225 |
| 11 | 216.65 | 22632 | 0.364 |
| 20 | 216.65 | 5475 | 0.088 |
| 32 | 228.65 | 868 | 0.013 |
| 47 | 270.65 | 110 | 0.0014 |
| 51 | 270.65 | 66.9 | 0.00086 |
| 71 | 214.65 | 3.96 | 0.000064 |
| 84.85 | 186.94 | 0.37 | 0.0000069 |

**大气分层**
- 对流层（0-11 km）：温度递减
- 平流层（11-47 km）：温度递增（臭氧层）
- 中间层（47-84 km）：温度递减
- 热层（84+ km）：温度急剧上升

### 3.2 空气动力学基础

**升力公式**
```
L = ½ × ρ × v² × S × CL
```

**阻力公式**
```
D = ½ × ρ × v² × S × CD
```

其中：
- ρ = 空气密度（kg/m³）
- v = 速度（m/s）
- S = 参考面积（m²）
- CL = 升力系数
- CD = 阻力系数

**动压（Dynamic Pressure）**
```
q = ½ × ρ × v²
```
- 火箭设计中的关键参数
- Max-Q：最大动压点，通常在10-15 km高度

### 3.3 阻力分析

**阻力组成**
```
CD = CD₀ + CDi + CDw
```
- CD₀：零升阻力（寄生阻力）
- CDi：诱导阻力（与升力相关）
- CDw：波阻（跨音速/超音速时显著）

**马赫数影响**
- 亚音速（M < 0.8）：阻力相对稳定
- 跨音速（0.8 < M < 1.2）：阻力急剧增加（音障）
- 超音速（M > 1.2）：形成激波，波阻主导

### 3.4 重力转弯（Gravity Turn）

**原理**
- 利用重力自然弯曲轨迹
- 减少主动转向所需的ΔV损失

**最优策略**
1. 垂直上升至100-200 m/s
2. 开始小幅倾斜（5-10°）
3. 让重力逐渐增加倾角
4. 到达目标高度时达到水平

**优点**
- 节省燃料（相比纯垂直+纯水平）
- 减少结构应力
- 自动控制简单

### 3.5 最大动压（Max-Q）

**定义**
- 飞行过程中动压的最大值
- 通常发生在速度增加但密度仍较高的阶段

**典型值**
- 航天飞机：~12 kPa
- SpaceX Falcon 9：~25 kPa
- Saturn V：~30 kPa

**工程意义**
- 结构设计的临界点
- 需要在此点前限制推力或速度
- 超过Max-Q后可以增加推力

## 4. 姿态控制与稳定

### 4.1 坐标系定义

**机体坐标系（Body Frame）**
- X轴：纵向（向前为正）
- Y轴：横向（向右为正）
- Z轴：垂直（向下为正，右手定则）

**惯性坐标系（Inertial Frame）**
- 原点：地心
- XY平面：赤道面
- X轴：指向春分点

**轨道坐标系（Orbital Frame）**
- R轴：径向（指向地心）
- T轴：切向（速度方向）
- N轴：法向（轨道面法线）

### 4.2 欧拉角

**三个旋转角**
1. **滚转（Roll, φ）**：绕X轴旋转
2. **俯仰（Pitch, θ）**：绕Y轴旋转
3. **偏航（Yaw, ψ）**：绕Z轴旋转

**旋转顺序**
- 常用：Z-Y-X（偏航-俯仰-滚转）
- 注意万向节锁（Gimbal Lock）问题

### 4.3 四元数（Quaternion）

**定义**
```
q = w + xi + yj + zk
```
其中 w² + x² + y² + z² = 1

**优势**
- 避免万向节锁
- 计算效率高
- 插值平滑（SLERP）

**旋转应用**
```
v' = q × v × q⁻¹
```

### 4.4 姿态控制方法

**反作用轮（Reaction Wheel）**
- 通过改变轮的转速产生扭矩
- 精度高，适合精细控制
- 会饱和，需要卸载

**控制力矩陀螺（CMG）**
- 高速旋转的陀螺仪
- 通过改变陀螺轴方向产生大扭矩
- 适合大型航天器

**推力器（Thruster）**
- RCS（反应控制系统）
- 快速响应
- 消耗推进剂

**气动舵面**
- 仅在大气层内有效
- 副翼、升降舵、方向舵
- 无燃料消耗

### 4.5 稳定性分析

**静稳定性**
- 重心（CG）在前，压力中心（CP）在后
- 类似箭羽的作用

**动稳定性**
- 阻尼效应
- 振荡衰减

**Margin of Stability**
```
Static Margin = (CP位置 - CG位置) / 参考长度
```
- 正值：稳定
- 负值：不稳定
- 典型值：0.05-0.15

## 5. 重力与引力场

### 5.1 牛顿万有引力定律

**基本公式**
```
F = G × (m₁ × m₂) / r²
```

其中：
- G = 6.67430 × 10⁻¹¹ m³/(kg·s²)
- m₁, m₂ = 两物体质量
- r = 距离

**重力加速度**
```
g = GM / r²
```

对于地球：
- GM = μ = 398600.4418 km³/s²
- 地表g₀ = 9.80665 m/s²

### 5.2 重力随高度变化

**精确公式**
```
g(h) = g₀ × (R / (R + h))²
```

其中：
- R = 地球半径 = 6371 km
- h = 高度

**近似公式（低高度）**
```
g(h) ≈ g₀ × (1 - 2h/R)
```

**典型值**

| 高度 | 重力加速度 | 相对地表 |
|------|-----------|---------|
| 0 km | 9.80665 m/s² | 100% |
| 100 km（卡门线） | 9.51 m/s² | 97% |
| 400 km（ISS） | 8.69 m/s² | 89% |
| 35786 km（GEO） | 0.224 m/s² | 2.3% |

### 5.3 非球形地球效应

**J2项（扁率效应）**
- 地球不是完美球体，赤道略鼓
- J2 = 1.08263 × 10⁻³

**影响**
- 轨道面进动（RAAN变化）
- 近地点旋转（Argument of Perigee变化）

**RAAN进动率**
```
dΩ/dt = -3/2 × J2 × (R/p)² × n × cos(i)
```

其中：
- p = a(1-e²) = 半通径
- n = √(μ/a³) = 平均运动
- i = 轨道倾角

**应用**
- 太阳同步轨道：选择特定倾角使轨道面跟随太阳
- 典型SSO倾角：~98°（逆行轨道）

### 5.4 多体引力问题

**限制性三体问题**
- 两个大质量天体 + 一个小质量物体
- 存在5个拉格朗日点（L1-L5）

**拉格朗日点**

| 点 | 位置 | 稳定性 | 应用 |
|----|------|--------|------|
| L1 | 两天体之间 | 不稳定 | 太阳观测卫星 |
| L2 | 外侧 | 不稳定 | JWST望远镜 |
| L3 | 对面 | 不稳定 | 理论研究 |
| L4 | 前方60° | 稳定 | 特洛伊小行星 |
| L5 | 后方60° | 稳定 | 特洛伊小行星 |

**希尔球（Hill Sphere）**
- 天体引力主导的区域
- 半径：`rH = a × (m/3M)^(1/3)`

## 6. 发射窗口与轨道转移

### 6.1 发射窗口

**定义**
- 允许发射的时间段
- 由目标轨道和发射场位置决定

**影响因素**
1. **轨道倾角约束**
   - 最小倾角 = 发射场纬度
   - 例如：卡纳维拉尔角（28.5°N）→ 最小倾角28.5°

2. **交会窗口**
   - 与空间站或其他航天器对接
   - 需要考虑相位角

3. **行星际窗口**
   - 霍曼转移的最佳时机
   - 例如：火星发射窗口每26个月一次

### 6.2 霍曼转移（Hohmann Transfer）

**原理**
- 最节能的双脉冲轨道转移
- 适用于共面圆轨道之间的转移

**转移轨道**
- 椭圆轨道，近地点在内轨，远地点在外轨
- 半长轴：`a_transfer = (r₁ + r₂) / 2`

**所需ΔV**
```
ΔV₁ = √(μ/r₁) × (√(2r₂/(r₁+r₂)) - 1)
ΔV₂ = √(μ/r₂) × (1 - √(2r₁/(r₁+r₂)))
```

**转移时间**
```
t_transfer = π × √(a_transfer³ / μ)
```

**示例：LEO到GEO**
- r₁ = 6571 km（200 km高度）
- r₂ = 42164 km（GEO）
- ΔV₁ ≈ 2.44 km/s
- ΔV₂ ≈ 1.47 km/s
- 总ΔV ≈ 3.91 km/s
- 转移时间 ≈ 5.3小时

### 6.3 双椭圆转移（Bi-elliptic Transfer）

**适用条件**
- 当r₂/r₁ > 11.94时，比霍曼转移更节能

**过程**
1. 第一次加速到非常高的远地点
2. 在远地点调整近地点
3. 在近地点减速进入目标轨道

**优势**
- 某些情况下ΔV更小
- 但转移时间更长

### 6.4 轨道平面变更

**所需ΔV**
```
ΔV = 2 × v × sin(Δi/2)
```

其中：
- v = 当前轨道速度
- Δi = 倾角变化量

**关键点**
- 在速度最低点进行（远地点）
- 60°以上的平面变更极其昂贵
- 最好从发射时就选择正确倾角

**示例**
- LEO速度：7.8 km/s
- 改变10°倾角：ΔV ≈ 1.36 km/s
- 改变28.5°倾角：ΔV ≈ 3.82 km/s（几乎等于整个轨道速度！）

### 6.5 交会与对接

**相位角**
```
α = n_chaser × t - n_target × t + α₀
```

**接近策略**
1. **同轨道面追赶**
   - 降低轨道以增加角速度
   - 追上后提升轨道

2. **Rendezvous序列**
   - 远距离导引（几百公里）
   - 近距离导引（几公里）
   - 最终接近（几十米）
   - 对接（几厘米/秒）

**安全考虑**
- 保持碰撞避免走廊
- 备份 abort 方案
- 相对速度限制

## 7. 热力学与再入物理

### 7.1 再入加热机制

**主要热源**
1. **激波压缩**（占90%以上）
   - 高速气体被压缩升温
   - 形成高温等离子体

2. **摩擦加热**
   - 边界层内的粘性耗散
   - 贡献较小

**驻点温度估算**
```
T_stagnation ≈ T_∞ × (1 + (γ-1)/2 × M²)
```

其中：
- T_∞ = 自由流温度
- γ = 比热比（空气≈1.4）
- M = 马赫数

**典型值**
- 返回舱再入：M ≈ 25
- 激波温度：> 10000 K
- 表面温度：1500-3000°C

### 7.2 热防护系统（TPS）

**类型**

1. **烧蚀材料**
   - 通过材料分解带走热量
   - 一次性使用
   - 例：阿波罗指令舱、神舟飞船

2. **辐射冷却**
   - 高温表面辐射散热
   - 可重复使用
   - 例：航天飞机隔热瓦

3. **主动冷却**
   - 内部流体循环
   - 复杂但高效
   - 例：火箭发动机喷管

**关键参数**
- 热流密度（W/m²）
- 总加热量（J/m²）
- 峰值加热时刻

### 7.3 再入轨迹

**弹道系数**
```
β = m / (CD × A)
```

其中：
- m = 质量
- CD = 阻力系数
- A = 参考面积

**影响**
- β大：穿透深，加热集中，减速快
- β小：减速平缓，加热分散

**再入走廊**
- 上界：跳过大气层（lift too much）
- 下界：过载/加热过大（dive too steep）
- 典型宽度：±1-2°

### 7.4 黑障现象

**成因**
- 高温电离气体形成等离子体鞘
- 阻挡无线电波传播

**影响频率**
- UHF/VHF：完全阻断
- 更高频率：部分衰减

**持续时间**
- 典型：3-5分钟
- 高度范围：80-40 km

**解决方案**
- 等待黑障结束
- 使用中继卫星
- 更高频率通信

## 8. 导航制导与控制

### 8.1 惯性导航系统（INS）

**原理**
- 使用加速度计和陀螺仪
- 积分得到速度和位置

**误差累积**
- 位置误差随时间增长
- 典型漂移：1-10 km/hour

**组件**
1. **加速度计**
   - 测量比力（specific force）
   - 需要减去重力

2. **陀螺仪**
   - 测量角速度
   - 积分得到姿态

### 8.2 GPS导航

**基本原理**
- 至少4颗卫星
- 测量信号传播时间
- 解算三维位置+时间

**精度**
- 民用：5-10 m
- 军用：1 m以内
- RTK：厘米级

**局限性**
- 需要视线（LOS）
- 高动态环境可能失锁
- 易受干扰

### 8.3 制导律

**比例导引（Proportional Navigation）**
```
a_command = N × Vc × λ̇
```

其中：
- N = 导航常数（3-5）
- Vc = 接近速度
- λ̇ = 视线角速率

**应用**
- 导弹制导
- 交会对接
- 着陆引导

**PID控制**
```
u(t) = Kp×e(t) + Ki×∫e(t)dt + Kd×de/dt
```

- P：比例项（当前误差）
- I：积分项（消除稳态误差）
- D：微分项（预测趋势）

### 8.4 自主导航

**星敏感器**
- 拍摄星空图像
- 识别恒星图案
- 确定绝对姿态
- 精度：弧秒级

**地平仪**
- 检测地球边缘
- 提供俯仰/滚转参考
- 简单可靠

**激光雷达（LiDAR）**
- 近距离高精度测距
- 用于着陆和交会
- 例：SpaceX Dragon对接

### 8.5 故障检测与容错

**冗余设计**
- 三重冗余（Triple Redundancy）
- 投票机制（Voting）
- 多数决原则

**故障模式**
1. **传感器故障**
   - 交叉校验
   - 异常值检测

2. **执行器故障**
   - 重新分配控制权限
   - 降级模式运行

3. **计算机故障**
   - 热备份切换
   - 看门狗定时器

**安全模式**
- 检测到严重故障时进入
- 停止机动
- 保持稳定姿态
- 等待地面指令
