# 真实太阳系快速参考

## 核心概念

### 场景布局
```
地球在中心 (0,0,0)
太阳在左侧 (-10, 0, 0) = 1 AU
其他行星围绕太阳轨道运行
```

### 真实比例
- **1 AU** = 149,597,870.7 km
- **1场景单位** = 0.1 AU = 1,495,978.7 km
- **太阳位置**:(-10, 0, 0)

## GUI控制

### 启用太阳系
```
太阳系模型
└── 启用太阳系
```

### 控制选项
- ✅ 启用太阳系 - 添加其他行星到场景
- ☑ 显示轨道 - 显示行星轨道路径
- ☐ 显示小行星带 - 显示火星-木星间小行星带
-  时间流速 - 0.1x 到 10x
- 🎯 聚焦行星 - 快速定位到行星

## 使用技巧

### 1. 查看内行星(水星、金星、火星)
- 直接旋转视角即可看到
- 距离地球3.87-15.24单位

### 2. 查看外行星(木星、土星)
- 使用"聚焦行星"功能
- 或使用时间加速(5-10x)等待它们靠近

### 3. 查看极远行星(天王星、海王星)
- 必须使用"聚焦行星"
- 建议开启轨道显示以定位
- 距离191-300单位,非常远！

### 4. 观察轨道运动
- 水星最快(4.15x速度)
- 海王星最慢(0.006x速度)
- 使用10x时间加速观察外行星运动

## 代码接口

### 初始化
```javascript
// 传递现有地球
solarSystem = new SolarSystemModel(scene, earth)
solarSystem.initSolarSystem()
```

### 控制方法
```javascript
// 启用/禁用
solarSystem.group.visible = true/false

// 显示/隐藏轨道
solarSystem.toggleOrbits(true/false)

// 显示/隐藏小行星带
solarSystem.toggleAsteroidBelt(true/false)

// 时间缩放
solarSystem.setTimeScale(1.0)  // 0.1 - 10.0

// 聚焦行星
solarSystem.focusOnPlanet('Jupiter', camera, controls)
```

- **效果**:照亮地球和所有行星

## 常见问题

### Q: 为什么看不到外行星？
A: 它们距离极远(52-300单位),使用"聚焦行星"功能

### Q: 行星运动太慢怎么办？
A: 调整时间流速到5-10x

### Q: 地球在哪里？
A: 始终在场景中心(0,0,0),不会移动

### Q: 太阳在哪里？
A: 在地球左侧(-10,0,0),距离1 AU

### Q: 轨道显示的是什么？
A: 行星围绕太阳的轨道路径(以太阳为中心)

## 行星特点

### 水星
- 最小行星
- 最快公转(88天)
- 温差最大(-180~430°C)

### 金星
- 最热行星(462°C)
- 逆向自转
- 浓厚CO2大气

### 地球
- 场景中心
- 唯一已知有生命
- 液态水存在

### 火星
- 红色星球
- 有极冠
- 可能有液态水

### 木星
- 最大行星(11.21倍地球)
- 大红斑风暴
- 79颗卫星

### 土星
- 美丽光环
- 密度小于水
- 82颗卫星

### 天王星
- 侧躺旋转(97.8°)
- 蓝绿色
- 27颗卫星

### 海王星
- 最远行星
- 风速最快(2100 km/h)
- 14颗卫星

## 小行星带

- **位置**:火星和木星之间
- **范围**:15-22单位(2.2-3.2 AU)
- **粒子数**:2000颗
- **颜色**:灰色
- **作用**:分隔类地行星和气态巨行星

## 性能提示

### 可见性距离
- < 20单位:容易看到
- 20-50单位:需要仔细寻找
- 50-100单位:需要聚焦或放大
- > 100单位:必须使用聚焦功能

### 渲染优化
- 禁用阴影提升性能
- 使用64x64几何体细分
- 2000颗小行星粒子

- 距离:天文单位(AU)
- 公转周期:地球年
