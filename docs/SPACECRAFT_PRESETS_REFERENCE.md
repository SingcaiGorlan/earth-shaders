## 💡 选择建议

### 🎯 任务类型推荐

**近地轨道任务 (LEO)**
- ✅ 推荐：神舟飞船、龙飞船、联盟号
- 特点：中等推力和燃料，适合短期任务
- 轨道高度：200-2,000 km

**空间站对接**
- ✅ 推荐：神舟飞船、龙飞船、联盟号
- 特点：精确的姿态控制，足够的RCS推进剂
- 关键参数：RCS推力 ≥ 0.7 kN

**深空探测**
- ✅ 推荐：旅行者号
- 特点：超高比冲，极低燃料消耗
- 适用：月球、火星、小行星带及更远

**天文观测**
- ✅ 推荐：哈勃望远镜
- 特点：稳定的姿态控制，充足的电力
- 关键：高Isp发动机用于轨道维持

**长期驻留**
- ✅ 推荐：国际空间站
- 特点：超大空间，充足电力，生命维持系统
- 注意：需要定期补给

## 🔧 自定义设计指南

### 设计你自己的航天器

#### 步骤1: 确定任务目标
- 去哪里？（近地轨道、月球、火星、深空）
- 做什么？（载人、货运、观测、探测）
- 多久？（几天、几个月、几年）

#### 步骤2: 选择基础模板
- 从6个预设中选择一个最接近的
- 或者从"探索者一号"默认配置开始

#### 步骤3: 调整关键参数

**质量分配公式：**
```
总质量 = 干质量 + 燃料质量
燃料质量 = 任务所需Δv / (发动机Isp × 9.81) × 干质量
```

**推力需求估算：**
```
最小推力 = 总质量 × 重力加速度 (9.81 m/s²)
推荐推力 = 最小推力 × 1.5 - 2.0
```

**电力需求估算：**
```
基本需求 = 2-5 kW (生命维持 + 通信)
实验设备 = 5-20 kW (根据任务而定)
总需求 = 基本需求 + 实验设备
```

#### 步骤4: 验证设计

**检查清单：**
- [ ] 总质量合理吗？（参考预设范围）
- [ ] 推力足够起飞吗？（推力 > 重量）
- [ ] 燃料够用吗？（燃烧时间 × 消耗率 ≤ 燃料质量）
- [ ] RCS有足够推进剂吗？（至少20 kg）
- [ ] 电力够用吗？（太阳能输出 ≥ 总需求）

## 📈 性能计算示例

### 示例1: 设计一个月球探测器

**任务需求：**
- 从地球轨道到月球轨道
- 需要Δv ≈ 4,000 m/s
- 干质量估计：2,000 kg

**计算过程：**

1. **选择发动机：** 液氢液氧发动机（Isp = 450 s）

2. **计算燃料需求：**
   ```
   使用火箭方程：Δv = Isp × g₀ × ln(m₀/mf)
   
   4,000 = 450 × 9.81 × ln(m₀/2,000)
   ln(m₀/2,000) = 4,000 / (450 × 9.81) = 0.906
   m₀/2,000 = e^0.906 = 2.475
   m₀ = 4,950 kg
   
   燃料质量 = 4,950 - 2,000 = 2,950 kg
   ```

3. **最终配置：**
   ```
   干质量:     2,000 kg
   燃料质量:   2,950 kg
   总质量:     4,950 kg
   发动机:     液氢液氧，推力 3,000 kN，Isp 450 s
   燃烧时间:   2,950 / 10 = 295 s
   ```

### 示例2: 设计一个火星着陆器

**任务需求：**
- 从火星轨道降落到表面
- 需要Δv ≈ 2,000 m/s
- 干质量估计：1,500 kg

**计算过程：**

1. **选择发动机：** 液氧甲烷发动机（Isp = 360 s）

2. **计算燃料需求：**
   ```
   2,000 = 360 × 9.81 × ln(m₀/1,500)
   ln(m₀/1,500) = 2,000 / (360 × 9.81) = 0.566
   m₀/1,500 = e^0.566 = 1.761
   m₀ = 2,642 kg
   
   燃料质量 = 2,642 - 1,500 = 1,142 kg
   ```

3. **最终配置：**
   ```
   干质量:     1,500 kg
   燃料质量:   1,142 kg
   总质量:     2,642 kg
   发动机:     液氧甲烷，推力 2,500 kN，Isp 360 s
   燃烧时间:   1,142 / 8 = 143 s
   RCS推进剂:  50 kg (用于精确着陆)
   ```
