## 🎯 题目设计原则

### 1. **难度梯度**

#### 选择题(基础)
- 考查基本概念和原理
- 选项设计包含常见错误
- 帮助学生识别易混淆点

**示例**:
```
逃逸速度与圆周速度的关系是:
A. v_escape = v_circular          ❌ 相等(错误)
B. v_escape = 2 × v_circular      ❌ 2倍(常见错误)
C. v_escape = √2 × v_circular     ✅ 正确
D. v_escape = v_circular / √2     ❌ 倒数(错误)
```

#### 填空题(记忆)
- 考查重要公式和常数
- 强化关键数据记忆
- 为计算题打基础

**示例**:
```
地球引力常数 μ = ____________ km³/s²
答案:398600.4418
```

#### 计算题(应用)
- 考查公式应用能力
- 训练数值计算技巧
- 模拟真实工程问题

**示例**:
```
计算高度为400km的圆轨道卫星的速度和周期.
(已知:R_earth = 6371 km, μ = 398600.4418 km³/s²)
```

#### 简答题(综合)
- 考查理论理解深度
- 培养分析表达能力
- 连接理论与实践

**示例**:
```
解释为什么国际空间站(ISS)中的宇航员会"失重"？
真的没有重力吗？
```

## 💡 答案解析特色

### 1. **详细步骤**

计算题提供完整解题过程:

```markdown
11. 计算高度为400km的圆轨道卫星的速度和周期.

解:
r = R_earth + h = 6371 + 400 = 6771 km

v = √(μ/r) = √(398600.4418/6771) ≈ 7.67 km/s

T = 2π√(r³/μ) = 2π√(6771³/398600.4418) 
  ≈ 5545 s ≈ 92.4 min
```

### 2. **易错点提醒**

标注常见错误:

```
⚠️ 注意:r是距地心距离,不是高度！
正确:r = R_earth + h
错误:r = h
```

### 3. **多种解法**

部分题目提供替代解法:

```
方法1:使用精确公式 g(h) = g₀ × (R/(R+h))²
方法2:使用近似公式 g(h) ≈ g₀ × (1 - 2h/R)
比较两种方法的误差
```

### 4. **物理意义解释**

不仅给出答案,还解释含义:

```
Static Margin = 0.3(偏大,可能过度稳定)
理想范围:0.05-0.15
过大会导致响应迟缓
```

```markdown
## 错题记录

### 题目信息
- **章节**:第1章 轨道力学基础
- **题号**:计算题13
- **日期**:2026-05-01

### 原题
计算从200km LEO到GEO的霍曼转移所需ΔV

### 我的答案
ΔV = 3.5 km/s ❌

### 正确答案
ΔV₁ = 2.44 km/s
ΔV₂ = 1.47 km/s
总ΔV = 3.91 km/s ✅

### 错误原因
- 忘记分别计算两次脉冲
- 单位换算错误

### 知识点
霍曼转移需要两次速度增量

### 反思
下次注意:
1. 明确转移过程(两次脉冲)
2. 仔细检查单位
3. 验算结果合理性
```
