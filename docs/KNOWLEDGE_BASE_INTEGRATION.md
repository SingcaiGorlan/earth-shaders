## 💡 使用指南

### 搜索功能
- **快捷键**: `Ctrl + F` 聚焦搜索框
- **输入关键词**: 实时过滤相关内容
- **清除搜索**: 按 `ESC` 或清空搜索框
- **匹配高亮**: 匹配的文本会以黄色高亮显示

### 导航功能
- **左侧目录**: 点击章节标题快速跳转
- **自动高亮**: 滚动时自动标记当前阅读位置
- **平滑滚动**: 点击目录项后平滑滚动到目标位置

### 公式复制
- **点击公式框**: 自动复制到剪贴板
- **视觉反馈**: 显示 "✓ 已复制" 提示
- **方便粘贴**: 可直接粘贴到笔记或文档中

## 📊 性能优化

### 加载优化
- 内联CSS：减少HTTP请求
- 延迟加载JS：DOMContentLoaded事件
- 无外部依赖：纯原生实现

### 渲染优化
- CSS transform：硬件加速动画
- will-change：提前告知浏览器优化
- 防抖处理：搜索输入优化

### 内存优化
- 事件委托：减少事件监听器数量
- DOM复用：避免频繁创建/销毁元素
- 及时清理：移除临时元素和监听器

## 🐛 已知问题

### 当前限制
1. **搜索功能**: 仅支持文本匹配，不支持正则表达式
2. **公式渲染**: 使用纯文本，未使用MathJax/KaTeX
3. **离线访问**: 需要网络连接（可改造为PWA）

### 浏览器兼容性
- ✅ Chrome/Edge (推荐)
- ✅ Firefox
- ✅ Safari
- ⚠️ IE11及以下不支持（使用了现代CSS/JS特性）
