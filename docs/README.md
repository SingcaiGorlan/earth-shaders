# Three.js Earth Shaders - 专业航天器模拟系统

一个基于Three.js的地球可视化项目，包含KSP风格的航天器模拟系统。

## ✨ 主要功能

###  地球可视化
- 真实地球纹理（白天/夜晚/云层）
- GLSL着色器效果
- 大气层散射模拟
- 动态光照和阴影

###  KSP游戏模式
- **专业化航天器配置面板** - 完整的参数设置界面
- **6个真实航天器预设** - 神舟、龙飞船、联盟号、ISS、哈勃、旅行者号
- **精细3D模型** - 10+组件，金属质感，环境反射
- **专业HUD仪表** - 轨道参数、飞船状态、导航球
- **实时物理计算** - 轨道力学、姿态控制、燃料消耗

###  太阳系模型 (v1.7.0 真实比例版)
- **场景扩展** - 在现有地球场景基础上添加其他行星
- **真实太阳系比例** - 基于天文单位（AU）的真实距离
- **专业GLSL着色器** - 8套专业着色器，与地球模型质量一致
- **程序化生成** - 无需外部纹理，实时生成表面细节
- **太阳特效** - 等离子体动画、太阳黑子、太阳耀斑、日冕
- **类地行星** - 程序化地形、大气散射、菲涅尔边缘光
- **气态巨行星** - 大气带纹理、湍流效果、木星大红斑
- **土星光环** - 三层环带结构、卡西尼环缝、粒子分布
- **真实天文数据** - NASA数据，精确到小数点后3位
- **动态轨道运动** - 开普勒定律模拟，真实公转周期
- **行星光环系统** - 土星、天王星光环
- **小行星带** - 火星-木星之间，2000颗粒子
- **可视化轨道** - 可开关的轨道路径
- **时间控制** - 0.1x-10x速度调节
- **相机聚焦** - 快速定位到特定行星

###  卫星轨道模拟
- TLE数据解析
- 实时轨道计算
- 多卫星追踪

## 🎮 KSP游戏模式使用指南

### 启动KSP模式
1. 打开浏览器访问应用
2. 在启动页面选择 "KSP游戏模式"
3. 点击 "启动" 按钮

### 打开配置面板
**方法1：** 按键盘 `C` 键  
**方法2：** 在右侧GUI点击 "🔧 航天器配置面板"

### 配置航天器
1. **选择预设模板** - 点击6个真实航天器之一
2. **微调参数** - 修改质量、推力、燃料等
3. **应用配置** - 点击 "✓ 应用配置" 按钮
4. **开始飞行** - 使用WASD控制航天器

### 飞行控制
- `W/S` - 增加/减少节流阀
- `A/D` - 左右滚转
- `Q/E` - 上下俯仰
- `T` - 开关SAS自动稳定
- `G` - 开关RCS姿态控制
- `V` - 切换相机模式
- `.` - 加速时间
- `,` - 减速时间
