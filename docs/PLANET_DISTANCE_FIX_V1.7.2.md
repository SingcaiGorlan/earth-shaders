# 行星距离比例修正 (v1.7.2)

## 问题分析

用户反馈"其他各行星之间的比例还是不对".

### 原因

之前的比例设置:
- **1 AU = 10单位** ❌
- 太阳位置: (150, 0, 0)
- 地球位置: (0, 0, 0)
- 地球到太阳实际距离: 150单位

这导致:
- 水星: 0.387 AU × 10 = **3.87单位**(太小)
- 金星: 0.723 AU × 10 = **7.23单位**(太小)
- 火星: 1.524 AU × 10 = **15.24单位**(太小)

**正确比例应该是:1 AU = 150单位**(与原有场景一致)

## 修正方案

### 修改 AU 比例常数

```javascript
// 修改前 ❌
const AU = 10  // Visualization scale

// 修改后 ✅
const AU = 150  // Scale to match existing scene (Sun at 150, Earth at 0)
```

### 小行星带修正

```javascript

// 修改后 ✅

const outerRadius = 3.2 * AU  // 480 units from Sun
```
