# 航天器专业化设计与配置面板说明

##  概述

本次更新对KSP游戏模式中的航天器进行了全面的专业化改造，参考真实航天器设计，创建了详细的3D模型和专业的配置面板。

## ✨ 航天器3D模型细化

### 组件结构

#### 1. **主燃料箱 (Main Fuel Tank)**
- **几何体**：圆柱体（半径0.15m，高1.2m）
- **材质**：金属银色（metalness: 0.8, roughness: 0.25）
- **细节**：
  - 3条蓝色条纹装饰
  - 高精度24边形截面
  - 环境反射增强（envMapIntensity: 1.5）

#### 2. **服务舱 (Service Module)**
- **几何体**：渐变圆柱体（上半径0.16m，下半径0.15m）
- **位置**：燃料箱上方
- **功能**：容纳电子设备、生命维持系统

#### 3. **返回舱 (Crew Capsule)**
- **几何体**：半球体（半径0.18m）
- **材质**：白色隔热材料
- **特色**：
  - 顶部半球形设计
  - 底部隔热罩（黑色，直径0.2m）
  - 模拟真实载人飞船返回舱

#### 4. **逃逸塔 (Launch Escape System)**
- **几何体**：细长圆柱（半径0.02-0.03m）
- **颜色**：红色警示色
- **整流罩**：锥形顶部
- **位置**：返回舱顶部
- **功能**：发射阶段紧急逃生

#### 5. **主发动机 (Main Engine)**
- **喷管**：
  - 渐缩渐扩喷管（bell nozzle）
  - 入口半径0.08m，出口半径0.14m
  - 高反射金属材质（metalness: 0.95）
  - 双面渲染（side: THREE.DoubleSide）
- **燃烧室**：
  - 圆柱形（半径0.1m，高0.15m）
  - 深灰色金属材质
- **喷管内部**：
  - 深色内衬（模拟耐高温材料）

#### 6. **太阳能板系统 (Solar Panel System)**
- **基板**：
  - 左右各一块（0.8m × 0.35m）
  - 深蓝色（#1a1a4e）
  - 发光效果（emissive: 0x000033）
- **网格线**：
  - 线框模式
  - 紫色半透明
  - 8×4网格精度
- **支架**：
  - 金属圆柱（半径0.015m）
  - 连接航天器主体
  - 高反射材质

#### 7. **天线系统 (Antenna System)**
- **天线杆**：
  - 细长圆柱（半径0.005-0.008m）
  - 银色金属
- **碟形天线**：
  - 半球形（半径0.06m）
  - 白色高反射
  - 双面渲染

#### 8. **RCS姿态控制推进器**
- **数量**：8组
- **分布**：
  - 前部4组（上下各2组）
  - 后部4组（上下各2组）
- **几何体**：小圆柱（半径0.02-0.03m）
- **火焰效果**：
  - 青色球体（半径0.015m）
  - 激活时显示
  - 半透明发光

#### 9. **发动机火焰效果**
- **外层火焰**：
  - 锥形（半径0.12m，高0.6m）
  - 橙色（#ff6600）
  - 半透明
  - 双面渲染
- **内层火焰**：
  - 锥形（半径0.06m，高0.4m）

#### 10. **标识和标记**

## ️ 配置面板功能

### 面板布局
配置面板采用专业科幻风格设计：
- **背景**：深色渐变 + 毛玻璃效果
- **边框**：青色发光边框
- **尺寸**：900px宽，最大85vh高
- **滚动**：自定义滚动条样式

### 功能分区

#### 1. **基本信息区**
- 航天器名称输入
- 航天器类型选择（下拉菜单）
  - 载人飞船
  - 货运飞船
  - 探测器
  - 空间站

#### 2. **质量参数区**
- 干质量输入
- 燃料质量输入
- 总质量显示（自动计算，只读）
- 实时联动：修改干质量或燃料质量时自动更新总质量

#### 3. **主发动机区**（橙色主题）
- 发动机类型选择
  - 液氧煤油发动机 (LOX/Kerosene)
  - 液氢液氧发动机 (LH2/LOX)
  - 固体火箭发动机 (Solid Rocket)
  - 离子发动机 (Ion Engine)
- 推力、比冲、燃烧时间、燃料消耗率输入

#### 4. **RCS姿态控制系统**（青色主题）
- 单组推力
- 比冲
- 推进剂质量

#### 5. **尺寸参数区**（橙色主题）
- 总长度
- 直径
- 太阳能板展宽

#### 6. **电力系统区**（绿色主题）
- 太阳能板输出
- 电池容量
- 当前电量

### 操作按钮

#### 1. **应用配置 (Apply)**
- 功能：读取所有输入值，更新航天器配置
- 效果：
  - 保存所有参数到spacecraftConfig
  - 更新物理参数（质量、燃料等）
  - 删除旧航天器模型
  - 使用新配置重新生成航天器
  - 显示成功通知（绿色）

#### 2. **重置默认 (Reset)**
- 功能：将所有输入恢复到默认值
- 效果：
  - 重置所有字段为初始值
  - 显示提示通知（橙色）

#### 3. **关闭 (Close)**
- 功能：隐藏配置面板
- 效果：切换面板显示/隐藏状态

## 🔧 使用方法

### 1. 打开配置面板
**方法1**：GUI控制
- 在KSP模式下
- 打开lil-gui面板
- 找到"🚀 KSP Game Mode"文件夹
- 点击" 航天器配置面板"按钮

**方法2**：代码调用
```javascript
kspGame.toggleConfigPanel()
```

### 2. 修改参数
- 在面板中修改所需参数
- 质量参数会自动计算总质量
- 所有参数都有合理范围限制

### 3. 应用配置
- 点击"✅ 应用配置"按钮
- 等待航天器重新生成
- 查看成功通知

### 4. 查看效果
- 3D模型会根据配置重新生成
- HUD显示更新后的参数
- 物理系统使用新质量计算

## 🚀 技术实现

### 3D建模
- **Three.js几何体**：
  - CylinderGeometry（圆柱、圆台）
  - SphereGeometry（球体、半球）
  - ConeGeometry（圆锥）
  - BoxGeometry（长方体）
  - PlaneGeometry（平面）

- **材质系统**：
  - MeshStandardMaterial（PBR材质）
  - MeshBasicMaterial（基础材质）
  - metalness/roughness参数
  - emissive发光效果
  - DoubleSide双面渲染

### 配置管理
- **配置对象**：
  ```javascript
  spacecraftConfig = {
    name, type,
    dryMass, fuelMass, totalMass,
    engine: { type, thrust, isp, burnTime, consumption },
    rcs: { thrust, isp, monopropellant },
    dimensions: { length, diameter, span },
    power: { solar, battery, current }
  }
  ```

- **数据流**：
  1. 用户修改输入
  2. 点击应用按钮
  3. 读取所有输入值
  4. 更新配置对象
  5. 删除旧模型
  6. 调用initSpacecraft()重新生成
  7. 显示通知

### 事件系统
- **自动计算**：干质量+燃料质量=总质量
- **输入验证**：最小值限制
- **联动更新**：修改相关参数自动同步

## 📝 注意事项

1. **性能考虑**：
   - 航天器模型较为复杂（约30+网格）
   - 建议在性能模式下简化模型

2. **参数范围**：
   - 所有参数都有最小值限制
   - 极端值可能导致物理异常

3. **配置持久化**：
   - 当前配置不保存到localStorage
   - 刷新页面后恢复默认值
   - 可在未来版本添加保存功能

4. **兼容性**：
   - 配置面板只在KSP模式下可用
   - 需要启用KSP模式才能打开
