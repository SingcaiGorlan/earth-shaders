# 🌌 完整太阳系模型 - 使用说明

## 📋 概述

本项目新增了一个完整的太阳系3D模型,包括太阳、八大行星、小行星带、轨道可视化等功能.所有参数基于真实天文数据,但为了可视化效果进行了适当缩放.

---

## ✨ 主要功能

### 1. **太阳系统** ☀️

#### 核心组件
- **太阳光球层** - 明亮的黄色球体,模拟可见光表面
- **内层日冕** - 半透明橙色光晕,模拟日冕内层
- **外层日冕** - 更扩散的橙红色光晕
- **日冕粒子** - 200个粒子组成的日冕物质抛射效果
- **点光源** - 照亮整个场景的主光源

#### 视觉效果
- 自发光的金色外观
- 多层光晕营造体积感
- 粒子系统模拟日冕活动
- 动态旋转动画

## 🎮 使用方法

### 启用太阳系模型

#### 方法1:GUI控制面板
1. 在右侧控制面板找到 "🌌 太阳系模型" 文件夹
2. 勾选 "启用太阳系" 复选框
3. 太阳系立即出现在场景中

#### 方法2:代码初始化
```javascript
// 在script.js中
solarSystemConfig.enableSolarSystem = true
initSolarSystem()
```

### 控制选项

#### 1. 启用/禁用太阳系
```
☑ 启用太阳系
```
- 开启时创建并显示太阳系
- 关闭时隐藏所有天体

#### 2. 显示/隐藏轨道
```
☑ 显示轨道
```
- 控制8条行星轨道线的可见性
- 不影响行星本身

#### 3. 显示/隐藏小行星带
```
☑ 显示小行星带
```
- 控制2000颗小行星粒子的可见性
- 位于火星和木星之间

#### 4. 时间流速调节
```
时间流速: [0.1 - 10.0]
默认值: 1.0
```
- **0.1x** - 慢动作,观察细节
- **1.0x** - 正常速度
- **5.0x** - 加速观察轨道运动
- **10.0x** - 快速演示

#### 5. 聚焦行星
```
聚焦行星: [下拉菜单]
选项: None, Mercury, Venus, Earth, Mars, 
      Jupiter, Saturn, Uranus, Neptune
```
- 选择行星后相机自动移动
- 视角定位到行星附近
- 方便观察特定行星

## 🔧 技术实现

### 数据结构

```javascript
planetData = [
    {
        name: 'PlanetName',           // 英文名
        nameCN: '行星中文名',         // 中文名
        radius: 1.0,                  // 相对半径
        distance: 20,                 // 轨道距离
        color: 0x6B93D6,             // 颜色(十六进制)
        orbitSpeed: 1.0,             // 轨道速度
        rotationSpeed: 0.02,         // 自转速度
        inclination: 0.0,            // 轨道倾角(度)
        eccentricity: 0.017,         // 轨道偏心率
        hasMoon: true,               // 是否有卫星
        hasRings: false,             // 是否有光环
        axialTilt: 0,                // 轴倾角(特殊情况)
        description: '描述文本'       // 中文描述
    }
]
```

### 比例缩放说明

#### 为什么需要缩放？
真实的太阳系比例太极端,无法直接可视化:
- 太阳直径是地球的109倍
- 海王星距离是地球的30倍
- 如果按真实比例,行星会变成看不见的点

#### 采用的缩放策略
1. **半径缩放:** 非线性缩放,突出大行星
2. **距离缩放:** 压缩距离,保持相对顺序
3. **速度缩放:** 加快慢速行星,便于观察

#### 缩放对照表

| 天体 | 真实半径(km) | 模型半径 | 真实距离(AU) | 模型距离 |

| 水星 | 2,439 | 0.38 | 0.39 | 10 |
| 金星 | 6,051 | 0.95 | 0.72 | 15 |
| 地球 | 6,371 | 1.00 | 1.00 | 20 |

### 材质系统

#### 类地行星(水星、金星、地球、火星)
```javascript
MeshStandardMaterial({
    color: planetColor,
    roughness: 0.8,    // 高粗糙度,岩石表面
    metalness: 0.2     // 低金属度
})
```

#### 气态巨行星(木星、土星)
```javascript
MeshPhongMaterial({
    color: planetColor,
    emissive: 0x111111,   // 微弱自发光
    specular: 0x222222,   // 镜面反射
    shininess: 10         // 低光泽度
})
```

#### 冰巨星(天王星、海王星)
```javascript
MeshPhongMaterial({
    color: planetColor,
    emissive: 0x112244,   // 蓝色自发光
    specular: 0x333333,
    shininess: 25
})
```

#### 月球
```javascript
MeshStandardMaterial({
    color: 0xCCCCCC,
    roughness: 0.9,    // 非常高粗糙度
    metalness: 0.1     // 几乎无金属
})
```

### 动画系统

#### 轨道运动
```javascript
// 使用极坐标计算位置
angle += orbitSpeed * deltaTime * timeScale
position.x = cos(angle) * distance
position.z = sin(angle) * distance
```

#### 自转运动
```javascript
// 绕Y轴自转
mesh.rotation.y += rotationSpeed * deltaTime
```

#### 月球公转
```javascript
// 相对于地球的局部坐标
moonAngle += moonSpeed * deltaTime
moon.position.x = cos(moonAngle) * moonDistance
moon.position.z = sin(moonAngle) * moonDistance
```

#### 小行星带旋转
```javascript
// 整体缓慢旋转
asteroidBelt.rotation.y += 0.0005 * deltaTime
```

## 📊 性能优化

### 1. 几何体优化
- **太阳:** 64×64细分(高质量)
- **行星:** 32×32细分(平衡质量)
- **月球:** 16×16细分(远距离)
- **轨道:** 128段(平滑曲线)

### 2. 粒子系统
- **日冕:** 200个粒子
- **小行星:** 2,000个粒子
- 使用BufferGeometry提高性能

### 3. 材质优化
- 共享材质实例
- 合理使用透明度和混合模式
- 避免过度使用高光反射

### 4. 渲染优化
- 阴影仅在必要时启用
- 轨道线使用Line而非Tube
- 光环使用RingGeometry而非自定义网格

## ❓ 常见问题

### Q: 为什么行星看起来这么大？
A: 为了可视化效果,行星尺寸被放大了.真实比例下行星会小到看不见.

### Q: 轨道是完美的圆形吗？
A: 不是.轨道使用了椭圆曲线,考虑了偏心率参数,但为了简化仍接近圆形.

### Q: 时间流速1x代表什么？
A: 1x是模型的基准速度,不代表真实时间.真实地球公转一周需要365天.

### Q: 可以添加更多行星吗？
A: 可以！修改`planetData`数组,添加新的行星对象即可.

### Q: 小行星带会碰撞吗？
A: 不会.当前版本中小行星只是视觉元素,没有物理碰撞检测.

### Q: 为什么只有地球有月球？
A: 为了性能和简化.未来版本会添加其他重要卫星.

### Q: 可以导出为视频吗？
A: 可以使用屏幕录制软件录制动画,或集成Three.js的视频导出功能.
