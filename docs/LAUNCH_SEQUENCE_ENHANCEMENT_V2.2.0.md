### 3. 🔥 多层发动机粒子系统 (Multi-Layer Engine Particles)

#### 粒子层级结构

**Layer 1: Core Flame（核心火焰）**
- **数量**: 300个粒子
- **颜色**: 白热→黄色（RGB: 1.0, 0.8-1.0, 0.3-0.6）
- **尺寸**: 0.25单位
- **速度**: 4单位/秒（最快）
- **范围**: 直径0.4单位，长度6单位
- **透明度**: 0.95（最亮）
- **混合模式**: AdditiveBlending
- **特点**: 最热、最亮的核心区域

**Layer 2: Main Flame（主火焰）**
- **数量**: 400个粒子
- **颜色**: 橙色→红色（RGB: 1.0, 0.3-0.6, 0.0）
- **尺寸**: 0.35单位
- **速度**: 3单位/秒
- **范围**: 直径0.8单位，长度8单位
- **透明度**: 0.85
- **混合模式**: AdditiveBlending
- **特点**: 主体火焰，可见度最高

**Layer 3: Outer Flame（外层火焰）**
- **数量**: 250个粒子
- **颜色**: 深橙/红（RGB: 1.0, 0.2-0.4, 0.0）
- **尺寸**: 0.45单位（最大）
- **速度**: 2.5单位/秒
- **范围**: 直径1.2单位，长度10单位
- **透明度**: 0.7
- **混合模式**: AdditiveBlending
- **特点**: 扩散的外层，增加体积感

**Layer 4: Thick Smoke（浓烟）**
- **数量**: 500个粒子（最多）
- **颜色**: 灰黑色（0x444444）
- **尺寸**: 1.0单位（可变0.5-2.0）
- **速度**: 1.8单位/秒（较慢）
- **范围**: 直径3单位，长度15单位
- **透明度**: 0.6
- **混合模式**: NormalBlending（不透明）
- **特点**: 真实烟雾效果，向外扩散

**Layer 5: Steam/Vapor（蒸汽）**
- **数量**: 200个粒子
- **颜色**: 白色（0xffffff）
- **尺寸**: 0.8单位
- **速度**: 1.2单位/秒（最慢）
- **范围**: 环形分布，半径2-6单位
- **透明度**: 0.4
- **混合模式**: AdditiveBlending
- **特点**: 水喷淋产生的蒸汽，向外扩散

**总计**: 1650个粒子同时渲染

#### 粒子动画特性

**湍流效果**
- Core: ±0.05随机偏移
- Main: ±0.08随机偏移
- Outer: ±0.10随机偏移
- Smoke: ±0.15随机偏移（最大扩散）

**蒸汽扩散**
- 基于极坐标的径向扩散
- 半径逐渐增大（+0.1/frame）
- 模拟真实蒸汽云团

**高度衰减**
- 随 altitude 增加降低透明度
- 公式：`opacityFactor = max(0.3, 1 - altitude / 200)`
- 200米高度时降至30%透明度
- 模拟大气稀薄效应

**效果描述**

- 分布在发射台周围（半径5-15米）

- 向下流动模拟喷水

**技术实现**
```javascript
const waterDeluge = new THREE.Points(geometry, material)
waterDeluge.position.set(6378, 0, 0) // 发射台位置
```

**动画**
- 持续向下运动（-2单位/秒）
- 触地后重置到顶部
- 循环播放

#### 发射台震动 (Launch Pad Shake)

**震动参数**
- 初始强度：0.5单位
- 衰减速率：0.98/frame
- 持续时间：5秒
- 方向：X和Z轴随机

**实现逻辑**
```javascript
shakeX = (Math.random() - 0.5) * intensity
shakeZ = (Math.random() - 0.5) * intensity
launchPad.position.x = 6378 + shakeX
launchPad.position.z = shakeZ
```

**视觉效果**
- 模拟火箭推力引起的地面震动
- 强度随时间指数衰减
- 5秒后完全停止

#### 发动机强光 (Engine Glow Light)

**光源参数**
- 类型：PointLight
- 颜色：橙红色（0xff6600）
- 初始强度：3.0
- 照射范围：50单位
- 位置：火箭底部下方2单位

**衰减动画**
- 2秒后开始衰减
- 每100ms降低5%强度
- 最低降至0.1
- 模拟远离后的光照减弱

### 5. 🎥 多视角摄像机系统 (Multi-View Camera System)

#### 四个摄像阶段

**Phase 1: 发动机特写 (0-3秒)**
- **位置**: 火箭侧下方（+8, -3, +8）
- **看向**: 发动机喷口（y - 5）
- **插值速度**: 0.05（快速响应）
- **效果**: 近距离观察点火瞬间
- **适用**: 感受发动机威力

**Phase 2: 侧面跟踪 (3-8秒)**
- **位置**: 侧面25单位外
- **高度**: 火箭上方5单位
- **插值速度**: 0.03（平滑跟随）
- **效果**: 完整火箭侧面视图
- **适用**: 观察整体上升姿态

**Phase 3: 仰视跟随 (<50米)**
- **位置**: 斜下方（+15, -30, +15）
- **距离**: 30单位
- **插值速度**: 0.02（缓慢稳定）
- **效果**: 从下往上仰望火箭
- **适用**: 感受火箭升空气势

**Phase 4: 标准跟随 (>50米)**
- **位置**: 斜上方（+25, +12, +25）
- **偏移**: 固定向量
- **插值速度**: 0.05（灵活跟随）
- **效果**: 经典第三人称视角
- **适用**: 长期轨道飞行观察

#### 摄像机切换逻辑

```javascript
if (missionTime < 3) {
    // Phase 1: Close-up
} else if (missionTime < 8) {
    // Phase 2: Side view
} else if (altitude < 50) {
    // Phase 3: Below view
} else {
    // Phase 4: Standard follow
}
```

**优势**
- 自动无缝切换
- 不同阶段最佳视角
- 电影级镜头语言
- 增强沉浸感

### 6. ⚡ 推力渐变系统 (Thrust Ramp-Up)

#### 推力曲线

**基础推力**: 15 m/s²

**渐变公式**
```javascript
thrustRampUp = min(missionTime / 2, 1)  // 2秒内从0到1
currentThrust = baseThrust * (0.5 + 0.5 * thrustRampUp)
```

**推力变化**
- T+0s: 7.5 m/s² (50%)
- T+1s: 11.25 m/s² (75%)
- T+2s: 15.0 m/s² (100%)
- T+2s+: 保持15.0 m/s²

**物理意义**
- 模拟发动机逐步达到全推力
- 避免突然加速的不真实感
- 更符合真实火箭发射
