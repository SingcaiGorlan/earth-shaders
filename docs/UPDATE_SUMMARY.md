#### 功能特点

2. **质量参数** - 干质量、燃料质量、总质量(自动计算)

4. **RCS系统** - 单组推力、比冲、推进剂量

6. **电力系统** - 太阳能输出、电池容量、当前电量

#### 操作按钮
- ✓ **应用配置** - 保存并应用当前设置
- ↺ **重置为默认** - 恢复到初始配置
- ✕ **关闭** - 取消更改并关闭面板

### 4. **智能配置管理系统** ⚙️

#### 自动计算
- **总质量** = 干质量 + 燃料质量(实时更新)
- **燃料消耗** = 推力 / (比冲 × 9.81)(物理公式)
- **加速度** = 推力 / 总质量(牛顿第二定律)

#### 参数验证
- ✅ 所有数值必须为正数
- ✅ 百分比值在 0-100 范围内
- ✅ 质量参数不能为零或负数
- ✅ 发动机参数在合理范围内

#### 通知系统
- **成功提示:** 绿色通知框
- **错误提示:** 红色通知框
- **信息提示:** 蓝色通知框
- **自动消失:** 2秒后淡出

## 📝 已知限制

### 当前版本
1. **配置不持久化** - 刷新页面后丢失自定义配置
2. **3D模型固定** - 配置不影响3D模型的视觉外观
3. **单一航天器** - 只能同时存在一个航天器
4. **无物理引擎** - 简化的物理计算,非完全真实
