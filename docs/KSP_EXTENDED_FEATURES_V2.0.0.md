## 🎯 游戏模式系统

### 模式定义
```javascript
mode: 'GROUND' | 'LAUNCH' | 'ORBITAL' | 'FREE_ROAM'
```

### 模式流转
```
GROUND (地面准备)
   ↓ startLaunchSequence()
LAUNCH (发射上升)
   ↓ altitude > 100km
ORBITAL (轨道飞行)
   ↓ 手动切换
FREE_ROAM (自由漫游)
```

### 各模式特性

#### GROUND Mode
- 显示航天中心场景
- 航天器位于发射台
- 等待发射指令
- 倒计时功能可用

#### LAUNCH Mode
- 发动机点火
- 垂直上升
- 粒子效果激活
- 重力转弯
- 摄像机跟随

#### ORBITAL Mode
- 轨道力学计算
- 预测轨迹显示
- 机动节点可用
- 完整飞行控制

#### FREE_ROAM Mode
- 原有自由漫游
- 无限制移动
- 相机自由控制
