# 三种应用模式说明

## 概述

本项目现已支持三种不同的应用模式,用户可以在启动页面选择合适的模式以获得最佳体验.

---

## 1. 教学演示模式 (Education Demo Mode)

**适用场景:** 课堂教学、科普演示、地球科学教育

### 特点
- ✅ **地球渲染** - 基础地球模型和纹理
- ✅ **云层系统** - 动态云层和大气效果
- ✅ **大气层** - 多层大气光晕效果
- ✅ **海洋波浪** - 动态海面波动效果
- ✅ **太阳系统** - 真实太阳和光照效果
- ✅ **极光效果** - 南北极光环可视化
- ✅ **热带气旋** - 台风/飓风模拟
- ✅ **卫星轨道** - 真实卫星追踪系统
- ✅ **时间控制** - 时间加速和回溯

### 禁用功能
- ❌ 太阳风粒子流
- ❌ 地球磁层可视化
- ❌ 引力场等势面
- ❌ 拉格朗日点
- ❌ KSP游戏控制

### 优势
- 视觉效果丰富,适合展示
- 包含气象现象(气旋、极光)
- 性能适中,流畅运行

**适用场景:** 航天器操控练习、轨道力学学习、太空探索游戏

### 特点
- ✅ **地球渲染** - 基础地球模型
- ✅ **云层系统** - 动态云层
- ✅ **大气层** - 大气光晕效果
- ✅ **太阳系统** - 光照效果
- ✅ **卫星轨道** - 卫星追踪
- ✅ **时间控制** - 时间缩放
- 🚀 **KSP控制系统** - 完整的航天器操控
  - 主发动机推力控制
  - RCS姿态控制系统
  - SAS自动稳定系统
  - 导航球(Navball)显示
  - 实时轨道参数显示
  - HUD抬头显示器

### 禁用功能
- ❌ 海洋波浪
- ❌ 太阳风粒子流
- ❌ 地球磁层
- ❌ 极光效果
- ❌ 引力场可视化
- ❌ 拉格朗日点
- ❌ 热带气旋

### 操作说明
- **W/S** - 俯仰控制
- **A/D** - 偏航控制
- **Q/E** - 滚转控制
- **Shift/Ctrl** - 节流阀增减
- **Z** - 切换SAS自动稳定
- **X** - 切换RCS姿态控制
- **空格** - 切换相机模式
- **T** - 时间加速
- **R** - 重置航天器

### 优势
- 沉浸式航天器操控体验
- 真实轨道力学模拟
- 专业飞行仪表显示

## 如何选择模式

### 选择教学演示模式,如果:
- 您是教师或学生
- 需要进行地球科学课堂演示
- 想要展示气象和自然现象
- 注重视觉效果和美观度

### 选择KSP游戏模式,如果:
- 您对航天器操控感兴趣
- 想学习轨道力学知识
- 喜欢互动式游戏体验
- 想要练习太空任务操作

### 选择科研模式,如果:
- 您从事空间科学研究
- 需要分析太阳风和磁层数据
- 研究引力场和轨道动力学
- 需要专业的科学可视化工具

## 技术实现

### 配置存储
用户选择的模式保存在 `localStorage` 中,键名为 `earthVizConfig`.

### 模式检测
在 `script.js` 中通过 `getAppMode()` 函数获取当前模式:
```javascript
function getAppMode() {
    return userConfig.appMode || 'education'
}
```

### 条件初始化
各功能模块采用条件初始化模式,根据模式自动启用/禁用相应功能:
```javascript
// 示例:仅在科研模式显示太阳风
solarWind.visible = isFeatureEnabled('solarWind', appMode === 'research')
```

### 预设配置
启动页面提供7种预设配置:
- ⚡ 最小配置
- ⚖️ 平衡模式
- 🚀 完整体验
- 🎮 性能优先
- 📚 教学模式
- 🚀 KSP模式
- 🔬 科研模式
