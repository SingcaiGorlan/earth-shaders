# 🚀 航天器发射模拟器使用指南

## 功能概述

本模块实现了一个简化的球形航天器模型，能够从地球表面任意位置发射，并在多体引力（地球、月球、太阳）作用下进行轨道运行模拟。

## 核心特性

### 1. 物理引擎
- **多体引力模拟**：基于真实物理的万有引力定律 (F = GMm/r²)
- **高精度积分**：使用四阶龙格-库塔法 (RK4) 进行数值积分
- **实时轨道计算**：动态计算航天器的位置、速度和加速度

### 2. 发射参数
- **发射地点**：可设置任意经纬度和高度
- **发射速度**：1000 - 15000 m/s 可调
- **发射方向**：方位角 (0-360°) 和仰角 (0-90°) 精确控制
- **时间控制**：支持暂停、加速/减速模拟

### 3. 可视化
- **球形航天器**：绿色发光球体表示航天器
- **轨道轨迹**：青色半透明线条显示飞行路径
- **实时状态**： altitude（高度）、speed（速度）等数据

## 使用方法

### 基本操作

1. **打开控制面板**
   - 在右侧GUI面板中找到 "🚀 Spacecraft Launcher" 部分

2. **设置发射参数**
   ```
   发射纬度 (°): -90 到 90 (默认: 28.5 - 肯尼迪航天中心)
   发射经度 (°): -180 到 180 (默认: -80.6)
   发射高度 (m): 0 到 100000 (默认: 0 - 海平面)
   发射速度 (m/s): 1000 到 15000 (默认: 7800 - 近地轨道速度)
   发射方位角 (°): 0 到 360 (默认: 90 - 向东)
   发射仰角 (°): 0 到 90 (默认: 0 - 水平发射)
   ```

3. **发射航天器**
   - 点击 "🚀 发射航天器" 按钮

4. **控制模拟**
   - ⏸️ 暂停模拟：勾选复选框暂停/继续
   - 时间缩放：0.1x 到 10x 调节模拟速度
   - 显示轨迹：切换轨道线显示/隐藏
   - 🔄 重置：将航天器恢复到发射前状态

## 技术细节

### 坐标系统

#### 地理坐标 → 笛卡尔坐标转换
```javascript
// 输入：纬度 φ, 经度 λ, 高度 h
// 输出：场景坐标 (x, y, z)

φ' = (90 - φ) × π/180  // 余纬度
λ' = (λ + 180) × π/180  // 经度弧度

r = R_earth × (1 + h/R_earth)

x = r × sin(φ') × cos(λ')
y = r × cos(φ')
z = r × sin(φ') × sin(λ')
```

#### 发射速度向量计算
```javascript
// 本地坐标系：北(N)、东(E)、天(U)
// 根据方位角 A 和仰角 ε 合成速度向量

v_horizontal = N × cos(A) + E × sin(A)
v_vector = v_horizontal × cos(ε) + U × sin(ε)
```

### 物理模型

#### 引力加速度计算
```javascript
// 牛顿万有引力定律
a = -GM/r² × r̂

其中：
- G = 6.67430×10⁻¹¹ m³/(kg·s²)  (万有引力常数)
- M = 5.9722×10²⁴ kg  (地球质量)
- r = 距离地心的距离
- r̂ = 指向地心的单位向量
```

#### RK4 数值积分
```javascript
// 四阶龙格-库塔法提高精度
k1 = f(t, y)
k2 = f(t + dt/2, y + k1×dt/2)
k3 = f(t + dt/2, y + k2×dt/2)
k4 = f(t + dt, y + k3×dt)

y(t+dt) = y(t) + (k1 + 2k2 + 2k3 + k4) × dt/6
```

### 性能优化

- **轨迹点限制**：最多保留500个轨迹点，避免内存溢出
- **自适应时间步长**：根据时间缩放自动调整
- **场景坐标转换**：统一使用场景单位，减少计算误差

## 常见问题

### Q1: 为什么航天器会坠回地球？
**A**: 如果发射速度低于第一宇宙速度（约7800 m/s），航天器无法维持轨道，会在引力作用下返回地面。

### Q2: 如何进入稳定轨道？
**A**: 
1. 确保发射速度 ≥ 7800 m/s（近地轨道）
2. 发射仰角设为 0°（水平发射）
3. 发射方位角建议设为 90°（向东，利用地球自转节省燃料）

### Q3: 如何实现星际转移？
**A**: 
1. 先进入近地停泊轨道（高度 ~200 km，速度 ~7800 m/s）
2. 在合适位置加速到第二宇宙速度（~11200 m/s）
3. 当前版本仅支持地球引力，未来可扩展多体引力

### Q4: 轨迹线为什么不显示？
**A**: 
- 检查 "显示轨迹" 复选框是否勾选
- 确保航天器已成功发射（isLaunched = true）
- 可能需要等待几秒钟积累足够的轨迹点

### Q5: 模拟速度太快/太慢怎么办？
**A**: 调整 "时间缩放" 滑块：
- < 1.0：减慢模拟速度，便于观察细节
- > 1.0：加快模拟速度，快速查看长期轨道演化

```javascript
// 在 spacecraftLauncher.js 的 calculateAcceleration 方法中
calculateAcceleration(position) {
    // 地球引力
    const earthAccel = this.earthGravityModel.calculateTotalGravity(position);
    
    // TODO: 添加月球引力
    const moonAccel = this.calculateMoonGravity(position);
    
    // 总加速度
    return earthAccel.add(moonAccel);
}
```

### 添加大气阻力
```javascript
// 简化大气模型
calculateDrag(velocity, altitude) {
    if (altitude > 100000) return 0; // 卡门线以上无大气
    
    const rho = 1.225 * Math.exp(-altitude / 8500); // 指数大气模型
    const Cd = 0.47; // 球形物体阻力系数
    const A = Math.PI * 0.05 * 0.05; // 迎风面积
    
    return 0.5 * rho * velocity.length()² * Cd * A;
}
```

### 添加推进系统
```javascript
// 简单的火箭发动机模型
applyThrust(thrust, duration) {
    const acceleration = thrust / this.mass;
    const direction = this.getForwardDirection();
    
    this.velocity.add(direction.multiplyScalar(acceleration * duration));
    this.mass -= this.fuelConsumptionRate * duration;
}
```
